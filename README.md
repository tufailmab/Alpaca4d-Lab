  <h1>Alpaca4d-lab</h1>

  <p>My learning repository for Alpaca4d, sharing Grasshopper workflows and example files for structural analysis experiments. I will be uploading here all the Grasshopper files I work on and analyze using alpaca4D in Rhino 8. Please note that these are practice and learning files created during my experiments, so accuracy is not guaranteed and they should be used for educational and exploration purposes only.</p>

  <h2>About Alpaca4d</h2>
  <p>
    <strong>Alpaca4d</strong> is a Grasshopper plugin built on <strong>OpenSees</strong>, providing tools for structural and earthquake engineering analysis. 
    This repository contains my personal learning files and workflows using Alpaca4d.
  </p>
